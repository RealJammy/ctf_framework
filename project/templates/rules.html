Some rules and what this is:


What is this?

This is a CTF created by myself, Euan. It is a jeopardy-style CTF, with a variety of challenges, including:

  • binary exploitation
  • reverse engineering
  • crypto
  • web
  • forensics
  • misc

    • The competition will be running for a certain period of time. The aim of a CTF is to gain flags and score points. The more points you score, the higher you climb up the leaderboard.
    To score points, you have to submit flags. Flags are just a string of characters, such as flag{sanity_check_fl4g}
    Flags are in the format flag{.*}


    Rules

    We can't let you just running around wild now, can we. Let's set out some rules so that this platform can (hopefully) run smoothly.

    1. Please don't bruteforce any flags. That takes the fun out of the CTF. It also means that you don't learn any new techniques
    2. Please don't share any flags. Flag sharing is prohibited, and will result in a ban from the platform.
    3. Don't attack the infrastrcuture
